Generally when an object is created it has a constructor ,this constructor points to js object, this js object has a proto null

Only Object constructor function and functions has an access to prototype property

In case of object constructor ,when new object is created with the use of object constructor.The real Object created
will search in its own object wheater the properties or methods are available ,If it is
not available it will search in Object constructor.This process is known as prototype chaining
